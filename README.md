# Spam_email_and_SMS_classifier
A lightweight, high-performance Machine Learning and Natural Language Processing (NLP) tool that classifies text messages as **Spam** or **Ham (Legitimate)** in real-time. Built to demonstrate fundamental NLP workflows, text vectorization, and probabilistic classification.

## Features
* **Real-time Inference:** Interactive command-line interface to type and test custom text strings instantly.
* **NLP Pipeline:** Implements tokenization and text feature extraction using Scikit-Learn's `CountVectorizer`.
* **Efficient ML Architecture:** Utilizes a `MultinomialNB` (Naive Bayes) classifier, optimized for text and word-frequency analytics.

## Tech Stack
* **Language:** Python 3
* **Libraries:** Scikit-Learn
